# 00.传统彩票与区块链彩票的对比
**传统彩票**
- 彩票会跑路
- 存在贪污腐败现象
**区块链彩票*
- 所有逻辑公开透明。因为所有代码都要上链，代码公开时需要verify。否则别人不会使用。
- 没有办法进行暗箱操作，不存在跑路。

# 01.区块链彩票业务分析
## 1.1 彩票规则
- 全民参与（play)
- 每一注只能投一个eth（可修改）
- 每个人可以投注多次
- 仅管理员定时开奖（kaijiang)
- 仅管理员可以退款（管理员到时间没有按时开奖，可以执行退奖）（tuijiang）
